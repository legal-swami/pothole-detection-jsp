# 🚧 Road Pothole Detection System

Java + JSP + OpenCV based web application to detect potholes from road images.

## 🚀 Live Demo
[https://pothole-detection-jsp.onrender.com]()

## ✨ Features
- Upload road images (JPG, PNG)
- Real-time pothole detection using OpenCV
- Simple web interface
- Works on mobile, tablet, laptop

## 🛠️ Tech Stack
- Java 11
- JSP / Servlets
- OpenCV 4.8
- Maven
- Tomcat 9

## 📦 Local Setup (Optional)
1. Install Java 11, Maven, Tomcat 9
2. Clone repo: `git clone https://github.com/your-username/pothole-detection-jsp.git`
3. Build: `mvn clean package`
4. Deploy WAR to Tomcat webapps/
5. Start Tomcat and open `http://localhost:8080/pothole-detection`
